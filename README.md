# User Management Dashboard

## Description
A React-based User Management Dashboard that allows users to view, add, edit, delete, search, sort, filter, and paginate user records using the JSONPlaceholder API.

## Features

- View Users
- Add User
- Edit User
- Delete User
- Search Users
- Sort Users
- Filter Users
- Pagination (10, 25, 50, 100)
- Loading Spinner
- Error Handling
- Responsive UI

## Technologies Used

- React
- Axios
- JSONPlaceholder API
- React Loader Spinner
- UUID

## Installation

```bash
npm install
npm start
```

## API

https://jsonplaceholder.typicode.com/users
